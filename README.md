# Blockspiel

Ein werbefreies Block-Puzzle-Spiel im Holzdesign -- speziell entwickelt für Senioren.

## Konzept

- **10x10 Raster** mit hochwertiger Holzoptik und Soundeffekten
- **Drag & Drop** -- Blöcke aus der Vorschau ins Spielfeld ziehen
- **Kein Zeitdruck** -- in Ruhe nachdenken und platzieren
- **Lokale Highscores** -- kein Account nötig, alles im Browser gespeichert
- **Werbefrei** -- keine Ablenkungen, keine Pop-ups

## Zielgruppe

Senioren, die ein ruhiges, vertrautes Puzzle-Spiel ohne störende Werbung und ständig wechselnde Oberflächen spielen möchten.

## Technik

- Reine HTML/CSS/JavaScript -- keine Frameworks, keine Build-Tools
- Progressive Web App (PWA) -- als App auf Tablet/Handy installierbar
- Responsive Design -- funktioniert auf Handy, Tablet und Desktop
- Offline-fähig -- spielt auch ohne Internet

## Lokale Entwicklung

Einfach `index.html` im Browser öffnen oder einen lokalen Server starten:

```bash
# Mit Python
python3 -m http.server 8000

# Dann öffnen: http://localhost:8000
```
